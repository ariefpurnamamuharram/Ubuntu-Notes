# Ubuntu-Notes
Where I write useful terminal commands in Ubuntu.

---

- Get CPU model name information.<br>
  `cat /proc/cpuinfo | grep "model name"`

- Get RAM information.<br>
  `free -h`
  
- Get Ubuntu version.<br> 
  `lsb_release -a`
  
- Get PHP version.<br> 
  `php -version`
  
- Get NginX version.<br> 
  `nginx -V`
  
- Get MySQL version.<br> 
  `mysql -V`
